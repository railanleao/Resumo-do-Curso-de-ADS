# Objetos de Fluxos
São os principais elementos gráficos para definir o comportamento de um processo de negócio. Eles são três:
| Elemento      | Notação               |
|---------------|-----------------------|
| Evento (Event)| https://pucminas.instructure.com/courses/87880/files/6616233/preview Desenho de um círculo |
| Atividade (Activity) | https://pucminas.instructure.com/courses/87880/files/6616197/preview Desenho de um quadrado |
| Gateway       | https://pucminas.instructure.com/courses/87880/files/6616249/preview Desenho de um losango |
# Dados
Os dados são representados com quatro objetos:
| Elemento  | Notação |
|----------| ----------|
| Objeto de dados (Data Object) | Desenho de um retângulo na vertical com a ponta da direita dobrada |
| Repositório de dados (Data Store) | Desenho de um cilindro |
# Objetos de Conexão
Conectam os Objetos de Fluxo entre si e entre outros elementos. São eles:
| Elemento | Notação |
| ------- | -------- |
|Fluxo de sequência
(Sequence Flow) |	Seta contínua para a direita |
|Fluxo de mensagens (Message Flor) | Seta tracejada para a direita |
|Associação	| Linha horizontal pontilhada ou Uma seta pontilhada para a direita |
